# Studio Bem Bonita

Site institucional desenvolvido para o Studio Bem Bonita, com foco em apresentar os serviços do salão, exibir trabalhos realizados, destacar transformações e facilitar o contato com clientes.

## Tecnologias

* HTML5
* CSS3
* JavaScript
* Google Fonts
* Font Awesome
* Google Maps Embed
* WhatsApp API

## Recursos do Projeto

* Hero section com identidade visual elegante e moderna
* Menu responsivo para desktop e dispositivos móveis
* Timeline interativa de serviços
* Galeria organizada por categorias
* Modal para visualização ampliada das imagens
* Comparador de Antes e Depois com controle deslizante
* Animações de entrada durante a navegação
* Mapa integrado com localização do estúdio
* Formulário com envio direto para WhatsApp
* Links rápidos para Instagram e WhatsApp
* Layout totalmente responsivo

## Estrutura

* `index.html` — Estrutura principal do site
* `style.css` — Estilos, animações e responsividade
* `script.js` — Interações, navegação e funcionalidades
* `img/` — Imagens da galeria, identidade visual e conteúdo do site

## Seções do Site

* Início
* Serviços
* Galeria
* Antes e Depois
* Sobre
* Contato

## Objetivo

O projeto foi desenvolvido para fortalecer a presença digital do Studio Bem Bonita, transmitindo profissionalismo, confiança e elegância. A proposta é oferecer uma experiência visual agradável, destacar os serviços oferecidos e facilitar o agendamento de clientes através dos canais de contato disponíveis.
